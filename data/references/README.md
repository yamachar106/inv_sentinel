# 参考書籍・資料

戦略のロジック根拠となる書籍データを格納するディレクトリ。
ここに格納されたデータは、シグナル判定ロジックの精緻化・パラメータ根拠・新戦略の実装に活用する。

## 格納形式
- Markdown (`.md`) or テキスト (`.txt`) でデータ化したものを配置
- ファイル名: `{戦略名}_{書名の略称}.md`
- 内容が長い場合は章ごとに分割可（例: `breakout_duke_ch1.md`）
